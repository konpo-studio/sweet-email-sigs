Changelog

    
What's new in Sweet & Simple Sigs

    

      
2026 · 05 · 28

      
    
        
  • Workspaces simplified. Dropped the passcode — anyone with the /w/<slug> link can browse and save. A prominent “Create workspace” card now greets you on the main editor when you're not in a team.
    • 
        
  • Logo size + better shapes. Logo-only signatures get a dedicated size (S / M / L / XL) and three display modes: Circle (round badge), Soft (natural aspect, rounded corners), Rectangle (natural aspect, sharp). Soft no longer crushes wide wordmarks into a tiny rounded square. (With a photo, the logo still shows as a small circular badge.)
    • 
        
  • Team workspaces. Create one at /new — pick a URL slug. Anyone with the /w/<slug> link can browse and save the team's signatures. No accounts.
    • 
        
  • Fix: non-square logos no longer offset to the top-left in the composite (pad-for-badge only applies when both avatar + logo are present).
    • 
        
  • Fix: Manifesto accent rule centers via align attr (Outlook-safe); Wildin' Out icons carry the border HTML attr.
    • 
        
  • Fix: avatar/logo could appear side-by-side with content in Gmail (some pipelines collapse adjacent <tr> rows). Switched the header wrapper to two independent tables — they can't be flattened and always stack.
    • 
        
  • Click the avatar/logo in the preview to upload, paste a URL, switch size/shape, or remove. Hover any social to reveal a + that adds another inline.
    • 
        
  • Inline edit in the preview. Click any text (name, title, company, email, website, etc.) to edit in place. Click a social icon/link to remove it or add another. Layout selection stays editor-only.
    • 
        
  • Single-character K placeholder logo (was the 4-dot mark). Cleaner default before users upload their own.
    • 
        
  • Editor reordered: Accent + Layout + Typography at the top under a Design band — fastest path to re-skin. Default seed populated with Konpo Studio (Instagram, LinkedIn, Substack) and socials shown as icons by default.
    • 
        
  • 3 more Experimental layouts: Terminal 04 (CLI prompt with window chrome), Manifesto 05 (newspaper masthead, huge serif), Receipt 06 (thermal-printer with ASCII rules and faux barcode).
    • 
        
  • New Experimental category in the Layout picker. First three: Field 01 (justified mono prose), Specimen 02 (giant CJK glyph anchor), Cargo 03 (dashed shipping tag).
    •
